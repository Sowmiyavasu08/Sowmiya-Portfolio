AI Summarizer

AI Summarizer is an AI-powered application designed to generate concise summaries from a wide range of document formats including PDFs, DOCX, TXT, PNG, JPG, and JPEG. It provides two options: extracting text from documents and summarizing the extracted content. The application is built using Python and Streamlit, with integration to OpenAI’s language models through Azure OpenAI. The tool offers a streamlined way for users—such as knowledge workers, students, and professionals—to quickly extract key insights from large volumes of content. The project leverages advanced natural language processing techniques to generate accurate and context-aware summaries.

    Technologies:
  • Python
  • OpenAI
  • AzureOpenAI
  • Streamlit
  • Flask
